Collatz-Problem
  1. Nimm eine natürliche Zahl n
  2. Wenn sie gerade ist: n/2
  3. Wenn sie ungerade ist: 3n+1
  4. fahre mit dieser Zahl bei 2 fort, wenn die Zahl nicht 1 ist
  5. Die Folge endet immer in der Schleife 1-4-2-1
  6. Problem: Ist das so? Beweise es!
  7. Erkenntnisse:
    1. Wenn die betrachtete Zahl eine 2er Potenz ist, dann führt das immer zu 1-4-2-1
    2. Eine 2er Potenz mit einem ungeraden Exponenten kann nicht durch 3n+1 erreicht werden.
    3. Starke Annahme: JEDE 2er Potenz mit einem geraden Exponenten kann durch 3n+1 erreicht werden.
    4. Zwei Beweismöglichkeiten, wenn die starke Annahme stimmt:
      1. Beweis, dass es eine Zahl gibt, die niemals eine 2er Potenz mit einem geraden Exponenten erreicht => Collatz liegt falsch
      2. Beweis, dass JEDE Zahl ZWANGSLÄUFIG bei einer 2er Potenz mit einem geraden Exponenten landet => Collatz liegt richtig
